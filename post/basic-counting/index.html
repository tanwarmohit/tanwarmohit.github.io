Basic Counting methods using Permutation and Combination techniques. These are my classroom notes, which might be helpful to others. It starts from basic fundamental principle of counting and then move to elementary Permutation, Combination and their applications. Each section includes some basic problems for practice and illustrative examples for more complex topics. Some problems contains many sub-problems which helps in getting better insights for the concept / theory explained.

Fundamental of Counting

Addition Rule

Let an event be occured through $Process_1$ in $m$ ways and through $Process_2$ in $n$ ways, then total number of ways in occuring an event is $m+n$.

There are $2$ cities $A$ & $B$. There are $2$ roads, $1$ train and $1$ flight connecting them. In how many ways, we can reach city $B$ from $A$?

NOTE

  • Addition means OR (option)
  • Number of ways should be mutually exclusive or non-overlapping.

Multiplication Rule

If an event $E_1$ can be occured in $m$ ways and another event $E_2$ can be occured in $n$ ways then $E_1$ and $E_2$ can be occured in $m*n$ ways.

There are 3 cities A,B & C. There are 2 roads, 1 train and 1 flight connecting A & B and there are 2 roads and 2 trains connecting B & C. In how many ways we can reach from A to C via B.

NOTE

  • Multiplication means AND (condition)
  • Events must be independent.
    • Events $E_1$ & $E_2$ are said to be independent when number of ways of occurrence of $E_1$ doesn’t depend or change w.r.t number of ways of occurrence of $E_2$ event.
      • For example: Find number of ways to reach from City A to City C via City B without using SAME mode of transport.
  • If between 2 processes, order is possible, then it will be considered by itself in multiplication.
    • Suppose we have 4 balls $B_1$, $B_2$, $B_3$, $B_4$ in a bag. In how many ways $2$ balls can be drawn? Hint: Think aloud in terms of Mutiplication Rule.
    • Suppose we have two bags $Bag_1$ and $Bag_2$ each having $4$ & $3$ balls respectively. In how many ways we can draw $1$ ball from each bag?

Bijection Rule

Number of favourable ways of occurrence of an event $E$ can be written in terms of total possible ways of occurrence of $E$ and unfavourable ways of occurrence of event of $E$ as: $n(E)$ = $n(T)$ - $n(E^{‘})$

  • This rule is helpful, when its practically very complicated to calculate all possible favourable ways of occurrence of $E$ rather than unfavourable ways.

Practice Problems

Warmup Problems

  1. If a coin is tossed $n$ times, then find possible number of outcome sequences.
  2. Find total number of ways in which $3$ objects out of $10$ different objects can be placed in a row on a shelf.
  3. Let $10$ different colour flags are given and a signal using 3 is to be made. Find total number of possible signals.
  4. Find total number of $5$ letter words using the letters of a word $TOUGH$, when
    • Repitition of letters / alphabets are allowed.
    • Repitition is NOT allowed.
  5. Let an objectve paper have $10$ questions with $4$ choices. Find possible number of answer sequences, if
    • Exactly $1$ option is correct
    • More than $1$ option is correct or in other words, $at least$ 1 option is correct.

Mega Problem

Let a $7$ digit number is to be made. Find total possible numbers if,

  1. All digits are different.
  2. Repetition of digits is allowed.
  3. No $two$ consecutive digits are same.
  4. All digits are $different$ and number formed is ODD.
  5. All digits are $distinct$ and number formed is EVEN.
  6. Number formed should NOT contain $4$ and
    • Repetition of digits are allowed
    • Repetition of digits are NOT allowed.
  7. Number formed should contain $5$ and
    • Repetition of digits are allowed
    • Repetition of digits are NOT allowed.
  8. Number formed should be divisible by $4$ and
    • Repetition of digits are allowed
    • Repetition of digits are NOT allowed.
  9. Number formed should be divisible by $3$, $0$ is not used and
    • Repetition of digits are allowed
    • Repetition of digits are NOT allowed.

Elementary Permutation and Combination

Permutation

It means $Arrangement$. > Illustration: In how many ways $ABC$ can arrange in a line? It means, we have $3$ places and we need to place $A$, $B$ & $C$ in those $3$ places.

Combination

It means $Selection$. > Illustration: We have $4$ objects. We need to choose $3$ objects. In how many ways we can do this? It means, we just need to focus on selecting the objects, not order in which they can be pick / selected.

  • If there are $n$ different objects and they need to be arrange in a line / row, in how many ways we can do this?
    • Solution: $n!$ hint: Think in terms of Multiplication rule.

Pratice Problems

  1. Find number of possible batting order in a team of $11$ players.
  2. How many more words can be formed using all letters of the word $TRIANGLE$?

Permutation & Combination Mix Problem Types

There are $n$ objects and we need to choose $r$ objects. In how many ways this selection can be done? $$C(n, r) = \frac{n!}{(n-r)!}$$

Sample Problems

  1. In how many ways $3$ letter words can be formed from a word $TRIANGLE$?
  2. In how many ways we can select team of $11$ players from $20$ players?
  3. How many triangles can be formed using vertices of $hexagon$?
  4. Find total number of 5 digits number in which digits are in
    • Ascending Order
    • Decending Order

There are $n$ different objects and $r$ to be selected and then we need to arrange them, or in other words, we need to arrange $r$ objects out of $n$ different objects. In how many ways we can do this? $$C(n,r).r! = \frac{n!}{(n-r)!}r! $$ $C(n,r)$ = Choose r objects and $r!$ = Arrange them

Sample Problem

  1. Out of $5$ different objects, $3$ needs to be arrange in an row. In how many we can arrange them?
  2. Find number of $4$ letter words formed using $a,e,i,o,p,q,r,s,t$ and
    • A letter can be used at most once
    • using exactly $2$ vowels
    • using atmost $1$ vowel

Let there are $n$ objects and its given that $n_1$ objects are of $type_1$, $n_2$ objects are of $type_2$ …. $n_k$ objects are of $type_k$ and rest all are of different types. What are the possible number of permutations? $$\frac {n!}{n_1!n_2!….n_k!}$$

Sample Problem

  1. Find number of ways of arranging $5$ $a$’s, $4$ $b$’s, $3$ $c$’s, $2$ $d$’s, $e,f,g,h,i,j$.
  2. Find how many ways we can rearrange letters of the word $MATHEMATICS$.

Grouping (Division & Distribution)

Division

  • Number of ways of dividing $n$ different objects into $r$ groups of group sizes, $n_1, n_2, n_3, ……, n_r$ where no two groups have same size, i.e. $n_i \ne n_j \forall i,j \in [1, r]$ $$\frac {n!} {n_1!n_2!….n_r!}$$

    • Solution Approach: $$C(n, n_1).C(n-n_1, n_2).C(n-(n_1 + n_2), n_3)….. C(n_r, n_r)$$
    • Illustration: Let there are $10$ different objects and they need to be divided into groups of sizes $3,2,5$. In how many ways we can distribute them?
      • Approach 1: $C(10, 5).C(5, 3).C(3,3) = \frac {10!}{5!5!} . \frac {5!}{3!2!} . 1$
      • Approach 2: All permutation = $10!$
        • $O_1O_2O_3O_4O_5|O_6O_7O_8|O9O{10}$
        • Think in terms of eliminating ordering of objects within the groups i.e. from all permutation we are reaching towards unique combination of objects within the groups.

  • Number of ways of dividing $n$ different objects into $r$ groups in which $k_1$ groups are of size $n_1$, $k_2$ groups are of size $n_2$ ….. $k_r$ groups are of size $n_r$ $$\frac {n!}{(n_1!)^{k_1}(n_2!)^{k_2}…(n_r!)^{k_r} . {k_1!k_2!…k_r!}}$$

    • Solution Approach
      • Lets take 5 objects $O_1, O_2, O_3, O_4, O_5$ and divide them in $3$ groups of size $1,2,2$.
      • Lets apply above method to divide $n$ objects in $r$ groups of different sizes
        • $[O_1], [O_2,O_3], [O_4, O_5]$
        • $[O_1], [O_4,O_5], [O_2, O_3]$
        • $[O_1], [O_2,O_4], [O_3, O_5]$
        • $[O_1], [O_3,O_5], [O_2, O_4]$
        • $[O_1], [O_2,O_5], [O_3, O_4]$
        • $[O_1], [O_3,O_4], [O_2, O_5]$
      • As you can see, above division of objects is repeated for groups of sizes $2$.
      • If we extend this all possible ways, we find that
        • $\frac {5!} {2!2!1!} . \frac {1} {2!}$

Distribution

  • Let $n$ different objects are to be distributed among $n$ people (one object to each) then number of ways: $$n!$$
    • $P_1, P_2, …, P_n$ persons are stand in line
    • Start placing objects $O_1, O_2, …., O_n$ in every possible permutation in front on each Person.
  • Let $n$ different objects are to be distributed among $r$ people with No restriction on distribution of objects but all objects should be distributed $$r^n$$
    • Consider objects $O_1, O_2,…. O_n$ to be distributed among $P_1, P_2, …, P_r$ persons, such that any person can get any number objects.
    • $r.r.r…..r$ ($n$ times)
  • Number of ways of distributing all $n$ different objects among $r$ people ($r > n$), such that no one gets more than $1$ object $$C(r, n). n! = \frac {r!}{(r-n)!} = r.(r-1).(r-2)……(r-(n-1))$$
    • Approach 1: First choose $n$ people out of $r$ people and apply distribution logic of distributing $k$ different objects to $k$ poeple.
    • Approach 2: $O_1$ can be assigned to $r$ people, $O_2$ can be assigned to $r-1$, and so on…
      • $r.(r-1).(r-2)….(r-(n-1))$
  • Let $n$ different objects are to be distributed among $r$ people with No restriction on distribution of objects to a person and No restriction on number of distributed objects but atleast one object needs to be distributed. Find all possible number of ways. $$(1+r)^n - 1$$
    • Solution Approach:1
      • Convert atleast one to exactly
      • exactly $1 OR$ exactly $2 OR$ …. exactly $n$ objects to be distributed among $r$ people without any restriction
      • ${n \choose 1}.r + {n \choose 2}.r^2 + ….. + {n \choose n}.r^n = (1 + r)^n - 1 $ … by binomial theorem
    • Solution Approach:2
      • Consider $1,2,3….r,Box$ $r + 1$ people and we need to distribute $n$ objects without restriction on number of objects given to a person.
      • $(1+r)^n - 1$
      • $-1$ because there would be a case when no object has been distributed to any person and all goes into box.
